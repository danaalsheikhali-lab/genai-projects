# Generative AI Projects

Applied generative-AI projects from Harvard Kennedy School **DPI-681 — *Public Problem Solving
with Generative AI*** (cross-registered from the Fletcher School). Each is a self-contained,
runnable notebook with its own README.

| Project | What it is | Most relevant to |
|---|---|---|
| [**AI Text Detector**](./ai-text-detector) | A classifier that distinguishes human vs. LLM-written news articles (heuristics → perplexity → TF-IDF), plus an adversarial-robustness analysis. | ML / model evaluation |
| [**RAG PolicyBot**](./rag-policybot) | A retrieval-augmented assistant grounded in policy-evidence RCTs, built to surface uncertainty and trade-offs. | Applied LLMs / RAG |
| [**AI Tutor + LLM-Judge Benchmark**](./llm-tutor-evaluation) | A Socratic tutor that never gives away answers, with an LLM-as-judge benchmark scoring it against a rubric. | LLM evaluation / alignment |

*Built on course-provided scaffolding with AI-assisted coding; all modeling and design decisions are my own.*
